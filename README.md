# Pokedex

A small client-side Pokedex that fetches Pokémon data from the public PokeAPI and displays the sprite.

## Features
- Search for a Pokémon by name (e.g., "pikachu")
- Displays the front sprite image
- Minimal, dependency-free HTML/CSS/JS

## Files
- `index.html` — main UI
- `script.js` — fetch logic and DOM updates
- `style.css` — simple styles
- `ReadMe.md` — this file

## Usage
1. Open `index.html` in your browser (no server required).
2. Enter a Pokémon name in the input and click "Fetch Pokemon".

## Notes
- Requires an internet connection to reach https://pokeapi.co.
- If a Pokémon name isn't found, errors are logged to the browser console. I can add on-screen error messaging if you want.
